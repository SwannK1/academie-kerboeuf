# Domaine 6 — Résolution de problèmes (CM2)

## Description

Ce domaine couvre la démarche de résolution de problèmes à une ou plusieurs étapes,
impliquant les quatre opérations, la proportionnalité, les mesures et la géométrie.
L'accent est mis sur la compréhension de l'énoncé, le choix de la stratégie et la vérification.

## Format attendu

- 1 séquence = 1 compétence ciblée
- Chaque séquence est rangée dans un sous-dossier portant le nom de la compétence
- Chaque séquence contient trois feuilles au format Markdown :
  - `feuille-1.md` — découverte / leçon
  - `feuille-2.md` — exercices guidés
  - `feuille-3.md` — entraînement autonome
- Aucun PDF fictif ne doit être créé
- Le fichier `manifest.example.json` illustre la structure JSON attendue pour chaque séquence

## Exemple de séquences à venir

- Résoudre un problème à deux étapes avec les quatre opérations
- Résoudre un problème de proportionnalité en contexte réel
- Résoudre un problème de mesure (périmètre, aire, durée)
- Choisir et justifier une stratégie de résolution
